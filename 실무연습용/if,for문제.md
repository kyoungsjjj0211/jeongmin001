✅ 실무 감각 있는 JAVA 연습문제 (if / for)

— 난이도: 초급 ~ 중급 

🔹 문제 1 — 로그인 검증 (if문)

회사 내부 시스템에서 입력한 ID와 Password가 일치하는지 검사하는 프로그램을 작성하시오.

저장된 ID: admin

저장된 PW: 1234

요구사항

사용자에게 ID와 PW를 입력받는다.

둘 다 맞으면 “로그인 성공”

ID만 틀리면 “아이디 오류”

PW만 틀리면 “비밀번호 오류”

둘 다 틀리면 “아이디와 비밀번호 오류”

🔹 문제 2 — 근무시간 초과 수당 계산 (if문)

직원의 하루 근무 시간이 주어졌을 때 급여를 계산한다.

기본 근무 시간: 8시간

초과 근무 수당: 기본 시급 * 1.5

기본 시급: 10,000원

입력 예: 근무시간(정수)
출력: 총 급여

🔹 문제 3 — 상품 재고 경고 메시지 (if문)

상품 재고량(stock)에 따라 다음 메시지를 출력하시오.

50개 이상: “재고 충분”

20~49개: “재고 보충 필요”

1~19개: “재고 위험”

0개: “품절!”

🔹 문제 4 — 주문 합계 계산 (for문)

고객이 구매한 상품 가격 배열이 다음과 같다:

int[] prices = {12000, 5000, 3000, 25000, 8000};


요구사항
for문을 사용하여 총 합계를 구하고 출력하시오.

🔹 문제 5 — 평균 별점 계산 (for문)

리뷰 별점 배열이 있다:

int[] stars = {5, 4, 3, 5, 4, 5};


for문을 사용하여 리뷰 평균을 계산하시오.
결과는 소수점까지 포함해서 출력할 것.

🔹 문제 6 — 최근 30일 로그인 횟수 중 특정 조건 탐색 (for + if)

다음은 최근 30일의 로그인 횟수 데이터이다:

int[] loginCounts = {2, 3, 0, 5, 1, 0, 7, 3, 4, 10, 2, 0, 5, 8, 6, 0, 1, 2, 3, 5, 9, 0, 2, 3, 1, 4, 6, 0, 3, 2};


요구사항

로그인 횟수가 5회 이상인 날의 개수를 세서 출력

로그인 횟수가 0인 날의 개수를 출력

가장 로그인 횟수가 높았던 날의 횟수를 출력

🔹 문제 7 — 사내 PC 관리: 비밀번호 변경 주기 알림 (for + if)

사원들의 비밀번호 사용일(며칠째 사용 중인지):

int[] passwordDays = {10, 45, 93, 30, 120, 14, 87};


요구사항
비밀번호 사용일에 따라 다음과 같이 출력:

90일 이상: “비밀번호 변경 필요”

30~89일: “안전”

0~29일: “최근 변경”

for문으로 모든 사원을 검사하여 출력하시오.

🔹 문제 8 — 결제 시스템에서 VAT 포함 가격 계산 (for)

상품 가격 배열이 있을 때 모든 상품에 대해 VAT 10%를 추가한 최종 가격을 출력하시오:

int[] items = {10000, 25000, 33000, 120000};


출력 예:

상품 1: 11000원
상품 2: 27500원
...
